# Codex

## Project Info

Sistema de operações para redes de fast food.

## Como rodar o projeto

1. Clone o repositório
2. Instale as dependências:
```sh
npm install
```
3. Inicie o servidor de desenvolvimento:
```sh
npm run dev
```

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Shadcn UI
- Supabase
